## Props
| **参数**  | **类型** |              **可选值**               | **默认值** |        **说明**         |
|:-------:|:------:|:----------------------------------:|:-------:|:---------------------:|
|  name   | string |                 -                  |    —    | svg 文件名 / Iconify 图标名 |
| flip  | string |    horizontal / vertical / both    |    -    |  翻转方式，支持水平、垂直以及同时翻转   |
| rotate | number |                  0 - 360                  |    0    |         图标旋转角度          |
|  color   | string |                 -                  |    -    |         图标颜色          |
|  size   | string / number |                 -                  |    -    |          图标尺寸           |
