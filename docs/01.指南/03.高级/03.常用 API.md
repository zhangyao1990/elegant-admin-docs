## 事件总线
基于 [mitt](https://github.com/developit/mitt) 简单封装，使用方法请查阅官方文档。

使用方式
```js
import eventBus from '@/utils/eventBus'
eventBus.on()
eventBus.emit()
eventBus.off()
```
通过on监听/emit触发/off移除事件
