框架设置了3 种导航栏模式

## 顶部菜单模式

```js
const globalSettings: Settings.all = {
  menu: {
    menuMode: 'head',
  },
}
```
## 侧边栏模式（含主导航）

```js
const globalSettings: Settings.all = {
  menu: {
    menuMode: 'side',
  },
}
```
## 侧边栏模式（无主导航）

```js
const globalSettings: Settings.all = {
  menu: {
    menuMode: 'single',
  },
}
```
